## Homework Instructions:

#### Sass Requirements:
Your project must use Sass and include:


* At least one variable file
* At least one mixins file
* At least one additional partial file
* Nested selectors where appropriate
* At least one custom mixin

#### Layout Requirements:
Use Both
* CSS Grid
* Flexbox

#### Responsive Requirements:

Your website must include responsive layouts for:

* Desktop
* Tablet
* Mobile
The mobile version must:

Display a navigation icon instead of the full navigation menu
Stack content into a single-column layout
Maintain readable typography and spacing
